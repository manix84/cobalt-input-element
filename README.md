## Basic Setup

- Clone repo: `git clone https://github.com/manix84/cobalt-input-element.git`
- Install dependencies: `npm i`

### Development

- Run `npm run dev`
- Open http://localhost:3000

### Production

- Run `npm run build`
- Run `npm run start`
- Open http://localhost:3000

## Testing

### Watch

- Run `jest --watch`

### Single Run

- Run `jest --ci`
